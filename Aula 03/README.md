# 🏫 Atividade Acadêmica: Star Game - ETEC

Este projeto foi desenvolvido como uma atividade prática para a disciplina de **PW2 (Programação Web 2)**, sob orientação da **Professora Celide**.

## 👤 Identificação
* **Série:** 2º Ano do Ensino Médio
* **Instituição:** ETEC
* **Disciplina:** Programação Web 2 (PW2)
* **Professor(a):** Celide

## 🎮 Sobre o Projeto
O objetivo da atividade foi criar a interface de uma plataforma gamer chamada **Star Game**. O foco principal foi o domínio de propriedades avançadas de CSS, utilizando o conceito de unidades flexíveis e layout moderno.

## 🚀 Desafios Técnicos Superados
Nesta aula de PW2, aplicamos:
* **Unidades de Medida:** Uso de `vh` e `vw` para o cabeçalho e `%` para os containers.
* **Layout com Flexbox:** Organização dos cards de missões para garantir alinhamento e fluidez.
* **Estilização Gamer:** Implementação de gradientes, sombras e efeitos visuais com CSS.
* **Resolução de Bugs:** Ajuste de seletores e correção de fluxo de elementos.

## 🛠️ Tecnologias
* HTML5
* CSS3
